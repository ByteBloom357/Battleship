# Sea Battle — C++ Console Game 

A classic Battleship implementation written in C++. This project demonstrates procedural programming techniques, manual and random ship placement logic, and interaction with the Windows Console API for a better user experience

## Features
- **Dynamic Board Interaction:** Real-time rendering of 10x10 game boards
- **Ship Placement Logic:** Supports both manual input and a randomized algorithm
- **Custom UI Colors:** Visual feedback for hits (Red), ships (Green), and missed shots using `Windows.h`
- **Session Timer:** Built-in timer to track game duration.

##  Technical Stack
- **Language:** C++
- **Environment:** Visual Studio
- **Key Concepts:** Multidimensional arrays, procedural logic, and time management

##  How to Play
1. **Setup:** Choose to place your ships manually or randomly
2. **The Fleet:** Includes ships ranging from 4-deck to 1-deck units.
3. **Combat:** Enter coordinates (row 0-9, column 0-9) to fire at the enemy
4. **Victory:** Sink all enemy ships to win!

##  Installation
1. Clone the repository.
2. Open the source file in **Visual Studio**.
3. Compile and Run (Ctrl + F5).
