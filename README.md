# Sea Battle — C++ Console Game 

A classic Battleship implementation written in C++. This project demonstrates procedural programming techniques, manual and random ship placement logic, and interaction with the Windows Console API for a better user experience[cite: 1].

## Features
- **Dynamic Board Interaction:** Real-time rendering of 10x10 game boards[cite: 1].
- **Ship Placement Logic:** Supports both manual input and a randomized algorithm[cite: 1].
- **Custom UI Colors:** Visual feedback for hits (Red), ships (Green), and missed shots using `Windows.h`[cite: 1].
- **Session Timer:** Built-in timer to track game duration[cite: 1].

##  Technical Stack
- **Language:** C++
- **Environment:** Visual Studio
- **Key Concepts:** Multidimensional arrays, procedural logic, and time management[cite: 1].

##  How to Play
1. **Setup:** Choose to place your ships manually or randomly[cite: 1].
2. **The Fleet:** Includes ships ranging from 4-deck to 1-deck units[cite: 1].
3. **Combat:** Enter coordinates (row 0-9, column 0-9) to fire at the enemy[cite: 1].
4. **Victory:** Sink all enemy ships to win![cite: 1]

##  Installation
1. Clone the repository.
2. Open the source file in **Visual Studio**.
3. Compile and Run (Ctrl + F5).
